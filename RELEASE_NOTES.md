## WA Auto Sender v1.0.0

Application Windows autonome pour l'envoi automatique de messages WhatsApp — 100% locale, bilingue FR/AR.

### Téléchargement

| Fichier | Description |
|---|---|
| **WA Auto Sender Setup 1.0.0.exe** | Installateur (recommandé) — crée un raccourci bureau |
| **WA Auto Sender 1.0.0.exe** | Version portable — double-clic, aucune installation |

### Installation rapide

1. Téléchargez et lancez l'un des fichiers ci-dessus
2. **Connexion** → scannez le QR code WhatsApp
3. **Contacts** → importez votre fichier de numéros (`.txt`, `.csv`, `.xlsx`)
4. **Modèles** → choisissez ou créez votre message
5. **Envoi auto** → cliquez **Démarrer**

### Fonctionnalités

- Envoi automatique avec cadence intelligente (intervalles, limite journalière, pauses)
- Interface Français + Arabe
- Import numéros marocains (06/07, +212)
- Modèles de messages business
- Suivi envoyé / en attente / échec
- Déconnexion et réinitialisation intégrées
- Chromium embarqué — rien à télécharger au premier lancement

### Prérequis

- Windows 10/11 (64 bits)
- Connexion Internet (pour WhatsApp uniquement)
